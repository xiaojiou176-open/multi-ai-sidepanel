Prompt packs

Start a compare run from reusable packs instead of a blank prompt.

Prompt Switchboard works best when you already know the task you want to compare. Prompt packs turn that idea into a repeatable starting point, so the side panel can move straight into the first useful compare run.

Writing Pack

Use this when you want models to rewrite, tighten, or reframe the same piece of text.

  • Rewrite this paragraph in a clearer, friendlier tone for a GitHub README.
  • Summarize this launch plan in three crisp bullets for a product update.

Recommended compare set: ChatGPT, Gemini, Perplexity.

Research Pack

Use this when you want side-by-side trade-offs, blind spots, and competing framings.

  • Compare the trade-offs between React and Vue for a browser extension UI.
  • Explain the strongest argument for and against local-first AI comparison workflows.

Recommended compare set: ChatGPT, Gemini, Perplexity.

Coding Explanations Pack

Use this when you want debugging explanations or architecture summaries in parallel.

  • Explain why a browser extension might fail during a persistent-context Playwright test run.
  • Explain this browser extension architecture to a new maintainer in plain English.

Recommended compare set: ChatGPT, Gemini, Grok.

Rewrite Pack

Use this when you want bilingual explanation, tone shifts, or product copy cleanup.

  • Rewrite this product description so a first-time visitor understands it in under 10 seconds.
  • Explain this feature in English first, then rewrite it in plain Chinese for a teammate.

Recommended compare set: ChatGPT, Gemini, Qwen.
