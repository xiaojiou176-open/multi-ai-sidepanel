# Code of Conduct

## Our Standard

This project is committed to a respectful, constructive, and inclusive
community.

Participants are expected to:

- be respectful and professional
- focus criticism on ideas, code, and process instead of people
- avoid harassment, discrimination, and personal attacks
- avoid posting secrets, tokens, cookies, private keys, or personal data

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- insulting, demeaning, or abusive language
- publishing private information without permission
- posting exploit details or sensitive material in public channels
- disrupting project discussion in bad faith

## Reporting

For behavior problems, open the smallest practical public report without
sensitive personal details and request a private handoff if needed.

For security-sensitive issues, use the process described in
[`SECURITY.md`](./SECURITY.md) instead.

## Scope

This code of conduct applies to repository issues, pull requests, discussions,
and other project-managed communication channels.
